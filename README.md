Sample - Superstore
# Objective 
This report provides a simplified analysis of Superstore retailer data to 
understand sales and profit performance across different categories. The 
analysis includes key metrics such as total sales, total profits, best and worst 
performing products, and sales breakdown by category.

# Tools Used 
1-Microsoft Excel
2-Pivot Tables
3-Charts
4-Data Cleaning
5-Power BI

#Dataset 
1️⃣ Row ID
A serial number for each row, used to distinguish between 
each sale.
2️⃣ Order ID
Order number; each order may contain more than one 
product.
3️⃣ Order Date
Order creation date.
4️⃣ Ship Date
Date of products being shipped to the customer.
5️⃣ Ship Mode
Shipping method used (Standard – First Class – Second 
Class).
6️⃣ Customer ID
Customer identification number in the system.
7️⃣ Customer Name
Name of the customer who purchased the product.
8️⃣ Segment
Customer Type:
Consumer
Corporate
Home Office
9️⃣ Country
Country (mostly USA in this file).
10️⃣ City
City where the order was delivered.
11️⃣ State
State.
12️⃣ Postal Code
Postal code.
13. Region
Geographic region within the country (East – West – Central 
– South).
14. Product ID
Product ID within the system.
15. Category
Major product category (Furniture – Office Supplies –
Technology).
16. Subcategory
Smaller subcategories within the category (e.g., Chairs –
Phones – Binders).
17. Product Name
Product name.
18. Sales
Final amount paid by the customer for the product (Price * 
Quantity).
19️⃣ Quantity
Number of units sold of the same product in the same order.
20️⃣ Discount
Discount percentage applied to the product.
21️⃣ Profit
Net profit on the transaction (can be positive + or negative -
).
